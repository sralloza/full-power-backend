# `release-changelog`

Releases a new version in the changelog.

**Usage**:

<div class="termy">
```console
$ python scripts/release-changelog.py [OPTIONS] NEW_VERSION
```
</div>

**Arguments**:

* `NEW_VERSION`: [required]

**Options**:

* `-c, --git`: Commits the result  [default: False]
* `--help`: Show this message and exit.
